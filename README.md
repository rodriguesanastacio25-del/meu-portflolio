<p>Desenvolvedor Front-End em formação</p># 💻 Meu Portfólio

Este projeto é um portfólio profissional desenvolvido como MVP (Produto Mínimo Viável), com o objetivo de apresentar minhas informações e projetos de forma simples, organizada e responsiva.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## 📌 Funcionalidades
- Estrutura semântica com header, sections e footer
- Layout responsivo para dispositivos móveis
- Exibição dinâmica de projetos utilizando JavaScript
- Interface simples e organizada

## 📱 Responsividade
O projeto foi desenvolvido seguindo o conceito de Mobile-First, garantindo uma boa experiência tanto em smartphones quanto em telas maiores.

## 🧠 Aprendizados
Durante o desenvolvimento deste projeto, apliquei na prática conceitos como:
- Organização de layout com Flexbox e Grid
- Manipulação do DOM com JavaScript
- Estruturação com HTML semântico
- Adaptação para diferentes tamanhos de tela

## 💼 Projetos

### 🔹 Portfólio Web
Site pessoal responsivo desenvolvido com HTML, CSS e JavaScript, com foco em apresentação profissional.

### 🔹 Lista de Tarefas
Aplicação simples para gerenciamento de tarefas, utilizando lógica em JavaScript e manipulação do DOM.

### 🔹 Landing Page
Página moderna com layout responsivo, focada em organização visual e experiência do usuário.
